## **WelearnCurriculumFinsh**和**WelearnTimeFinsh**
本脚本仅供学习参考交流，请使用者于下载24小时内自行删除，不得用于商业用途，否则后果自负。
## 简介
* 本人完全小白源码乱的一批就不发源码了
* 随时可能删库跑路
* 输入账号密码或cookie登录并一键完成课程  
* 可以选择课程
* 可指定每章节正确率 （多线程学习，同时学习整个单元)
* 已完成和未开放的章节会自动跳过   (可以指定时间或设置随机时间)
